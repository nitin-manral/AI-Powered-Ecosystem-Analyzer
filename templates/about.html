What is it?

The system simulates an IoT-based environmental network where daily readings from air, soil, water and forest sensors are collected and analyzed. Instead of showing raw numbers in a table, the dashboard converts them into meaningful insights: trends, alerts and predictions.

Goals

  • Provide a single view of ecosystem health across multiple indicators.
  • Detect anomalies early using rule-based alerts.
  • Estimate future air quality using a machine learning model.
  • Present data in a clean, easy-to-understand dashboard.

Under the Hood

Technology Stack

  • Python (Flask) for backend and routing.
  • Pandas for data handling & preprocessing.
  • Scikit-Learn Random Forest model for AQI prediction.
  • Matplotlib for generating trend charts.
  • HTML, CSS and a bit of JS for the front-end UI.

Design Choices

  • Light & dark mode for comfortable viewing.
  • Card-based layout for quick scanning of metrics.
  • Simple animations to make the dashboard feel alive.
  • Responsive layout that works on different screen sizes.

About This Project

MCA project on AI-based environmental monitoring and decision support.

This website is part of the AI Powered Ecosystem Analyzer project. It uses environmental data and an AI model (Random Forest) to:

  • Monitor daily environmental parameters
  • Analyze AQI and pollution risks
  • Generate alerts (High Temperature, Low Humidity, High AQI)
  • Predict AQI using Machine Learning
  • Display daily reports summarizing ecosystem conditions

Tools & Technologies

  • Python, Flask
  • Pandas, NumPy
  • Scikit-Learn (Random Forest)
  • Matplotlib
  • HTML, CSS

Future Scope

  • Integration with real IoT sensors for live data
  • Mobile app for citizens and field officers
  • Cloud deployment (AWS / Azure) for scalability
  • Deep learning based advanced forecasting
